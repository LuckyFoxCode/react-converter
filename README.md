## Documentation:

| Term           | Description                  |
| -------------- | ---------------------------- |
| **Day:**       | General daytime hours        |
| **Night:**     | General night hours          |
| **NightDays:** | Number of night days         |
| **TaxDay:**    | Rate per hour of day shift   |
| **TaxNight:**  | Rate per hour of night shift |
